Detailed Description

Class containing the necessary information for a CommandBuffer::drawIndexedIndirect command. Should be filled and uploaded to a buffer (or directly written to through a shader) and used for the DrawIndexedIndirect command. DrawIndexedIndirect allows to draw primitives using an IndexBuffer to select vertices/.
