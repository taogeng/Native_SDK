142 /*!****************************************************************************************************************
143 \brief API graphics pipeline wrapper. A GraphicsPipeline represents the configuration of almost the entire RenderState,
144 including vertex description, primitive assembly, Shader configuration, rasterization, blending etc. Access through
145 the Framework managed object GraphicsPipeline.
146 A ParentableGraphicsPipeline is a pipeline that is suitable to function as the "Parent" of another pipeline, helping
147 to create efficient Pipeline Hierarchies.
148 \description ParentableGraphicsPipelines can and should be used to make switching between different pipelines more
149 efficient. In effect, a ParentableGraphicsPipeline allows the user to create another (non-parentable pipeline) as
150 a "diff" of the state between the Parentable pipeline and itself, making the transition between them very efficient.
151 ******************************************************************************************************************/

154  friend class GraphicsPipeline_;
