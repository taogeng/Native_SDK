Detailed Description

Vulkan Default FBO (FBO pointing to the Back Buffer). This object is necessary for rendering anything to the screen. Should be used through the Fbo object. If a GLES direct manipulation is required, use through the DefaultFboVulkan Reference counted Framework object.
