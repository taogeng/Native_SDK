163 /*!****************************************************************************************************************
164 \brief The PowerVR Shell (pvr::Shell) is the main class that the user will inherit his application from.
165 \description This class abstracts the platform for the user and provides a unified interface to it. The user will
166  normally write his application as a class inheriting from the Shell. This way the user can have
167  specific and easy to use places to write his code - Application start, window initialisation, per
168  frame, cleanup. All platform queries and settings can be done on the shell (set the required Graphics
169  API required, window size etc.). Specific callbacks and queries are provided for system events
170  (keyboard, mouse, touch) as well as a unified simplified input interface provided such abstracted
171  input events as "Left", "Right", "Action1", "Quit" across different platforms.

385  \description This event abstracts, maps and unifies several input devices, in a way with a mind to unify several
386  platforms and input devices.

388  Left Click at Center, Space key, Enter key, Touch at Center cause Action1.

395  ******************************************************************************************************************/

397  {
