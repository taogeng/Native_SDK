template<char8 C1Name, char8 C2Name, uint8 C1Bits, uint8 C2Bits>
class pvr::assets::GeneratePixelType2< C1Name, C2Name, C1Bits, C2Bits >

Use this template class to generate a 2 channel PixelID.

Description:
Use this template class to generate a 2 channel PixelID (64-bit identifier for a pixel format used throughout PVR Assets from the channel information. Simply define the template parameters for your class and get the ID member. EXAMPLE USE:
uint64 myPixelID = GeneratePixelType2<'r', 'a', 8, 8>::ID;
Parameters
C1NameThe Name of the 1st channel (poss. values 'r','g','a','l',0)
C2NameThe Name of the 2nd channel (poss. values 'r','g','a','l',0)
C1BitsThe number of bits of the 1st channel
C2BitsThe number of bits of the 2nd channel
