Detailed Description

This class contains function pointers for all EGL extensions that existed at the time of publishing. Can be updated as required. The function pointers get automatically populated on application start during State Machine creation. (function pvr::createNativePlatformContext). Use as a namespace (e.g. eglext::WaitSyncKHR).
