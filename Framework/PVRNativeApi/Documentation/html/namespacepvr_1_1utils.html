Gets the openGL/ES equivalent texture format values, as per the Khronos KTX specification: http://www.khronos.org/opengles/sdk/tools/KTX/file_format_spec/.

Parameters
[in]pixelFormatThe pixel format (RGBA8888 etc)
[in]colorSpaceThe colorspace (srgb/lrgb)
[in]dataTypeThe datatype (Float32, unsignedbyte...)
[out]glInternalFormatSet to The internal format (a GLenum of the specified format GL_RGBA8888_sRGB)
[out]glFormatSet to The format (a GLenum of the specified format GL_RGBA)
[out]glTypeSet to The type (a GLenum of the specified format GL_FLOAT, GL_UNSIGNED_SHORT_1_5_5_5_REV)
[out]glTypeSizeSet to the number of components in the format
[out]isCompressedFormatSet to true if the format is compressed, false otherwise
Returns
true on success, false if a suitable type cannot be matched
