Detailed Description

template<typename IntegerType>
class pvr::NormalisedInteger< IntegerType >

Implementation of a Normalised Integer (an integer representing a fixed point value from 0..1). Contains functions to construct from Integer values, to convert to and from floating point values, arithmetic operators and similar.
