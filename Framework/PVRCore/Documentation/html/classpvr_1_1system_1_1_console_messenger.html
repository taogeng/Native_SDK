Detailed Description

The ConsoleMessenger is an implementation of Messenger that outputs to the Console window.

Description:
The ConsoleMessenger is an implementation of Messenger that outputs to the Console window. Additionally, if a Debug window is detected (usually on Windows), the Messenger will try to use that. Otherwise, if a Console is detected, it will output there. Lastly, in platforms where this is practical (i.e. not sandboxed)it will output to a text file called log.txt.
