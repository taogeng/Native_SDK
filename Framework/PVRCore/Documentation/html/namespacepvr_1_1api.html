Detailed Description

Main PVRApi Namespace. In the PVRCore projects, contains interfaces and enumerations that conceptually belong to PVRApi but need to be shared with other PVR Framework projects (GraphicsContext interface, enums)

typedef RefCountedResource< impl::Buffer_ > pvr::api::Buffer

Buffer Object.

Forwared-declared reference-counted handle to a Buffer. For detailed documentation, see PVRApi module.

Remarks
Default constructor returns an empty handle that wraps a NULL object. Use the IGraphicsContext's createBuffer to construct a Buffer. As with all reference-counted handles, access with the arrow operator.
