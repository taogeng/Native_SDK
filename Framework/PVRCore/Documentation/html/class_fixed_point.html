Detailed Description

template<typename IntegerType, bool IsSigned>
class FixedPoint< IntegerType, IsSigned >

ICC30 s15Fixed16Number representation for 32-bit signed fixed values, and u16Fixed16Number for 32-bit unsigned fixed values.

Description:
Always uses unsigned values for its storage type. Uses sign, bias and magnitude for signed values.
Template Parameters
IntegerTypeThe representation type.
IsSignedThis instance is signed.
