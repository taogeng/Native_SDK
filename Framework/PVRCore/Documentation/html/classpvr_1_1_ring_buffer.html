Detailed Description

template<typename>
class pvr::RingBuffer< typename >

A classic, templated, efficient RingBuffer implementation. Supports operations at both ends (front, back) and dynamic resizing when full. Does not at current implementations support iterators.

Template Parameters
ItemTypeThe type of the items stored in the RingBuffer
