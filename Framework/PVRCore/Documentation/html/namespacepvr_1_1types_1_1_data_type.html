bool pvr::types::DataType::isNormalised ( Enum  type)
inline

Return if the format is Normalized (represents a range between 0..1 for unsigned types or between -1..1 for signed types)

Parameters
typeThe format to test.
Returns
True if the format is Normalised.
Description:
A Normalised format is a value that is stored as an Integer, but that actually represents a value from 0..1 or -1..1 instead of the numeric value of the integer. For example, for a normalised unsigned byte value, the value 0 represents 0.0, the value 127 represents 0.5 and the value 255 represents 1.0.
